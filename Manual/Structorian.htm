Structorian Manual

Written by Dmitry Jemerov <yole@yole.ru>

The Structorian manual consists of two sections:

  • Introducing Structorian provides the overall description of Structorian and introduces the general concepts used by the program. Read this if you are just starting to use Structorian.
  • Structorian Script Reference provides comprehensive reference information on the syntax of Structorian Script (*.strs) files, field types and expressions.
